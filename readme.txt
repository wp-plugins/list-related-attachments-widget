Listed Related Attachments is a widget that will display attachments related to the current page/post

== Description ==

Listed Related Attachments is a small wordpress widget that will list all related attachments for the current post or page. The widget has options to filter the list by type of attachment such as image, ms word file, pdf, etc. Feel free to use and hack this plugin as you see fit.
The widget options are: title, number of attachments to display and type of attachment to display.

== Installation ==

1. Install the Listed Related Attachments widget
2. Activate the widget
3. Configure `Listed Related Attachments` widget as you need
